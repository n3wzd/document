사용자 입력을 받기 위해서는 `input()` 함수를 사용합니다. 이 함수는 사용자가 키보드로 입력한 문자열을 반환합니다. `input()` 함수를 호출할 때 인자로 전달된 문자열은 사용자에게 보여질 프롬프트 메시지입니다. 사용자 입력은 기본적으로 문자열로 처리됩니다. 

```
name = input("이름을 입력하세요: ")
print("안녕하세요,", name)
```
