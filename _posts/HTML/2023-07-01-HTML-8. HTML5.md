`HTML5`는 HTML의 최신 표준입니다.

HTML5에선 다음이 허용됩니다.
- 태그 이름 대문자 가능
- 속성값 따옴표 생략 가능
- 속성값 생략 가능
- 빈 태그 종료 태그(/) 생략 가능

- DOCTYPE 선언
```
<!DOCTYPE html>
```

- 문자셋 선언 (HTML5의 기본 문자 인코딩 방식은 UFT-8입니다.)
```
<meta charset="UFT-8">
```
